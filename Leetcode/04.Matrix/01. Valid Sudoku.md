**ðŸŽ¯ Task**:  
Determine if a 9x9 Sudoku board is valid based on the following rules:  
1. Each row must contain the digits `1-9` without repetition.  
2. Each column must contain the digits `1-9` without repetition.  
3. Each 3x3 sub-box must contain the digits `1-9` without repetition.  

**Note**: Empty cells are denoted by `"."` and can be ignored.  

---

## ðŸŒŸ Solution 1: Using a Flat Representation  

### Approach:  
1. ðŸŸ¢ Iterate through each cell in the board.  
2. ðŸ”„ If the cell is not empty (`.`), generate three validation tuples for:  
   - The row containing the number.  
   - The column containing the number.  
   - The 3x3 sub-box containing the number.  
3. âœ… Check for duplicates using a set.  
4. â±ï¸ **Time Complexity**: \(O(81) = O(1)\), as the board size is constant.  

### âœ¨ Implementation:  

```python
class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
        res = []
        for i in range(9):
            for j in range(9):
                elm = board[i][j]
                if elm != '.':
                    res += [(i, elm), (elm, j), (i // 3, j // 3, elm)]
        return len(res) == len(set(res))
```


## ðŸŒŸ Solution 2: Using Hashmaps for Rows, Columns, and Boxes

### Approach:  
1. ðŸŸ¦ Use three dictionaries (rows, cols, boxes) to track numbers in rows, columns, and sub-boxes.
2. ðŸ”„ Iterate through the board:
    - If the cell is not empty (.), check if the number already exists in the corresponding row, column, or box.
    - If it exists, return False.
    - Otherwise, add it to the corresponding dictionaries.
3. âœ… Return True if no conflicts are found.
4. â±ï¸ Time Complexity: O(81)=O(1). 

### âœ¨ Implementation:  

```python
from collections import defaultdict

class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
        rows = defaultdict(set)
        cols = defaultdict(set)
        boxes = defaultdict(set)
        
        for r in range(9):
            for c in range(9):
                if board[r][c] == ".":
                    continue
                
                # Check if the number already exists in the current row, column, or box
                if (board[r][c] in rows[r] or 
                    board[r][c] in cols[c] or 
                    board[r][c] in boxes[(r // 3, c // 3)]):
                    return False
                
                # Add the number to the corresponding row, column, and box
                rows[r].add(board[r][c])
                cols[c].add(board[r][c])
                boxes[(r // 3, c // 3)].add(board[r][c])
        
        return True
```
