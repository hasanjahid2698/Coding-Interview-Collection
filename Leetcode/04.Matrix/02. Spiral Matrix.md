**ðŸŽ¯ Task**:  
Given an \( m * n \) matrix, return all elements of the matrix in spiral order.  

---

## ðŸŒŸ Solution: Layer-by-Layer Traversal  

### Approach:  
1. ðŸŸ¢ Define boundaries for traversal:  
   - `top` and `bottom` for rows.  
   - `left` and `right` for columns.  
2. ðŸ”„ Traverse the matrix in four directions within the boundaries:  
   - **Left to right**: Traverse the topmost row.  
   - **Top to bottom**: Traverse the rightmost column.  
   - **Right to left**: Traverse the bottommost row (if rows remain).  
   - **Bottom to top**: Traverse the leftmost column (if columns remain).  
3. âœ… After each direction, update the corresponding boundary.  
4. â±ï¸ **Time Complexity**: \(O(m * n)\), as each element is visited once.  

---

### âœ¨ Implementation:  

```python
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        if not matrix:
            return []

        rows, cols = len(matrix), len(matrix[0])
        top, bottom, left, right = 0, rows-1, 0, cols-1
        result = []
        
        while len(result) < rows * cols:
            # Traverse from left to right along the top boundary
            for i in range(left, right+1):
                result.append(matrix[top][i])
            top += 1
            
            # Traverse from top to bottom along the right boundary
            for i in range(top, bottom+1):
                result.append(matrix[i][right])
            right -= 1
            
            # Traverse from right to left along the bottom boundary (if rows remain)
            if top <= bottom:
                for i in range(right, left-1, -1):
                    result.append(matrix[bottom][i])
                bottom -= 1
            
            # Traverse from bottom to top along the left boundary (if columns remain)
            if left <= right:
                for i in range(bottom, top-1, -1):
                    result.append(matrix[i][left])
                left += 1
        
        return result
```
